정의부분으로 가기 첫 페이지로 돌아가기

목록 만들기 연습

ul(unorder list), ol(order list), dl(definition list)

ul : 순서 없는 목록, 항목 앞에 블릿기호가 표시됨

  • java
  • oracle
  • jdbc
  • html5






ol : 순번이 매겨지는 목록

  1. html5
  2. css3
  3. javascript
  4. jquery
  5. web api





맨 위로 가기

dl : 제목과 순번을 하나로 묶어 목록을 만듦

정의부분
html5
W3C에서 발표한 웹 표준 2.0의 기술을 제공하는 새로운 웹페이지 제작용 마크업 언어이다.
css3
html5와 함께 웹 표준 2.0의 웹페이지의 스타일을 적용하기 위한 스타일시트 새 버전이다.
javascript
자바로 만든 웹페이지에 사용되는 스크립트 언어이다.
jquery
jquery 사에서 자바스크립트 소스를 작성해서 제공한 오픈 소스들을 말한다.
